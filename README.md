# User Service based on Event Sourcing

## Prerequisites
* docker 
* docker-compose
* .NET Core SDK 3.1.0

## How to run
* Set docker-compose project as StartUp project
* Run in your IDE

## UI
* Swagger: http://localhost:656/swagger/index.html
* MongoDB: http://localhost:8081/

# What was used
## Tech Stack:
* ASP.NET Core
* Docker
* Docker Compose
* MediatR
* Swagger
* MSSql
* MongoDb

## Principles:
* SOLID
* REST
